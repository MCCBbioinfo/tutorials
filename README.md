## How These Tutorials Are Hosted

These tutorials are written as R Markdown (`.Rmd`) documents and knitted to static
HTML. The rendered HTML pages are published with [GitHub Pages](https://pages.github.com/)
directly from this repository ([`MCCBbioinfo/tutorials`](https://github.com/MCCBbioinfo/tutorials/settings/pages)),
so each `*.html` file is served at:

```
https://mccbbioinfo.github.io/tutorials/<TutorialName>.html
```

To add a new tutorial, knit your `.Rmd` to `.html`, commit both files to the `main`
branch, and the page will be available at the URL above once GitHub Pages rebuilds.
